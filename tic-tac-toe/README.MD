# 🎮 Tic-Tac-Toe Game (Python)

A simple command-line **Tic-Tac-Toe** game built using **Python** and **Object-Oriented Programming (OOP)** concepts.

Players take turns placing **X** and **O** on a 3×3 board. The game automatically checks for a winner, detects a draw, validates user input, and allows players to play multiple rounds.

---

## 📌 Features

- ✅ Two-player gameplay
- ✅ Interactive command-line interface
- ✅ Object-Oriented Programming (Classes & Methods)
- ✅ Automatic winner detection
- ✅ Draw detection
- ✅ Input validation
- ✅ Prevents overwriting occupied positions
- ✅ Option to play again after the game ends
- ✅ Clean and readable board display

## 🎲 Game Rules

- Player **X** always starts first.
- Players take turns entering a position from **1 to 9**.
- Each number corresponds to a position on the board.

Example board:

```
 1 | 2 | 3
-----------
 4 | 5 | 6
-----------
 7 | 8 | 9
```

- A player wins by placing three identical symbols in:
  - Any row
  - Any column
  - Any diagonal

- If all positions are filled and no player wins, the game ends in a **Draw**.
